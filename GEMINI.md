## Project

Mockly is a web-based interview preparation platform where students practice role-specific interview questions and get their answers evaluated by a local LLM (Llama 3 via Ollama) with personalized feedback, hybrid scoring, and visual progress tracking.

### Core Value
Empower students to improve their interview performance through highly analytical, local hybrid evaluation (keyword, vector embedding, and LLM) and structured, actionable feedback with a simulated placement gating flow.
<!-- GSD:project-end -->

<!-- GSD:stack-start source:STACK.md -->
## Technology Stack

- **Frontend**: React (Vite), Tailwind CSS, Recharts, Axios
- **Backend**: Node.js, Express, Mongoose
- **Database**: MongoDB
- **Local AI**: Ollama running Llama 3 8B (default) and Ollama embeddings API
- **Auth**: JWT (JSON Web Tokens), bcrypt for password hashing
<!-- GSD:stack-end -->
